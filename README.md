# Project: Tic‑Tac‑Toe Game (Two‑Player)

A console‑based two‑player Tic‑Tac‑Toe game with move validation, win/draw detection, replay option, and persistent score history.  
Developed as a team project by 6 members, each responsible for a distinct module.

---

## Features

- **Interactive 3×3 board** – displayed after every move  
- **Two‑player turn management** – X and O alternate automatically  
- **Move validation** – prevents overwriting occupied cells and accepts only 1‑9 inputs  
- **Win & draw detection** – correctly identifies 3‑in‑a‑row (row, column, diagonal) and full board draws  
- **Replay option** – play multiple rounds without restarting the program  
- **Score history** – each round’s result (winner/draw) is saved to a file and can be viewed at the end
